	- ***一级降压***
		我简要的将一级降压分成几点来讲解
		1. 降压原理
			我选用的芯片是$MP4583$通过查阅手册得知 拓扑结构为降压式 降压原理与$LDO$不同 基本上可以理解为**开关斩波 + 储能滤波** 首先通过芯片内部的$MOS$管 当它闭合时 高压直接通入$PVCC$通向电感 电流逐渐增大 电感将电能储存起来 之后芯片反馈到输出电压足够 输入被切断 但我们都知道 电感的电流不允许突变 将之前存储的能量释放 最后就是斩波 电感送来的电流是一阵阵的 可以形如海浪 我们可以添加大电容来滤波 具体芯片是如何反馈到输出电压足够 那么就到了第二点
		2. 降压公式
			输出的最终电压，完全是由接在$FB$引脚上的两个分压电阻决定的
			对于这类开关稳压芯片，其内部都有一个精准的参考基准电压
			$$V_{FB} = 0.8V$$
			所以我们可以简单的算出
			$$V_{out} =  V_{FB} \times \left (1 + \frac{R_top}{R_bottom} \right) = 0.8V \times \left (1 + \frac{1000}{191} \right) \approx 4.988V$$
			我在$R_top$旁并联了一个小小电容 这个电容的作用是在电机瞬态发力 电流急剧下降 类似于被抽走的情况下 此电容可将输出端电压微小的下降传递给$FB$引脚 让芯片更快做出响应 提升系统动态响应速度
			
	- ***二级降压***	
		二级降压很简单 我选择了$MP20051$ 就是一个很简单的$LDO$可以分成几点来说
		1. 降压公式 与一级降压相同$$V_{out} =  V_{ref} \times \left (1 + \frac{R_top}{R_bottom} \right)$$
		2. 可以看到 我画了俩一样的一个是$AVCC$一个是$DVCC$ 
		 -  $DVCC$
			   - 专门供给数字电路 由于数字信号翻转很快 会产生很多的高频噪声 滤是滤不掉
		- $AVCC$
			 -  模拟信号非常脆弱。如果和数字电路共用一个电源，数字噪声会直接通过电源线串入模拟端，导致电机电流采样波动、磁编码器角度跳变
	 
	 - ***隔离地平面***
		隔离也是比较重要的一Part 在电机控制系统中，电流的***质量***或者说***优劣程度***是完全不同的，如果混在一起会出大问题
